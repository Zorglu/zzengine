public static get(string $name[, string $method = "" ][, string $filter = 0 ][, array<string|int, mixed> $filter_options = ["flags" => FILTER_NULL_ON_FAILURE] ]) : mixed
Parameters
$name : string

Nom du parametre

$method : string = ""

Nom de la méthode utilisée. Si la méthode est vide, alors elle n'est pas prise en compte. Sinon la méthode devra être la bonne. Ex : si le parametre ID est demandé avec la methode POST mais qu'il a été transmis avec la methode GET celui ci ne sera pas valide

$filter : string = 0

nom du filtre à utiliser pour la validation Ex : FILTER_VALIDATE_EMAIL pour que le parametre corresponde bien au format email Ex : FILTER_VALIDATE_INT pour un entier valide

$filter_options : array<string|int, mixed> = ["flags" => FILTER_NULL_ON_FAILURE]

des Parametres complémentaires pour le filtrage Ex : ["flags" => FILTER_NULL_ON_FAILURE, "options" => ["min_range" => 1] pour que le parametre fournis soit Sup à 0
